### 1. 公开披露渠道
- **13F 文件** - 机构持仓，每季度披露，45天延迟

### 3. 更及时的信息源
- **期权异常交易** - Unusual Whales、FlowAlgo
- **国会议员交易** - QuiverQuant、CapitolTrades
- **暗池数据** - FlowAlgo、BlackBoxStocks

- 机构持仓（个股页面 → 机构）

## 自动交易系统

### Moomoo OpenAPI
- 官方提供 Python SDK
- 支持行情获取、下单、持仓查询
- 文档：https://openapi.moomoo.com
- 需要申请开通权限，可能有资金量要求

### 美股自动交易要求

**监管层面：**
- **PDT规则**：账户 < $25,000 时，5个交易日内不能超过3次日内交易

**技术层面：**
- 稳定网络和服务器（推荐云服务器）
- API 延迟考虑（美国服务器更快）
- 异常处理机制

### 券商API对比

| 券商 | API友好度 | 特点 |
|------|----------|------|
| Interactive Brokers (IBKR) | 最强 | 专业级，功能全 |
| Alpaca | 很好 | 免佣，专为量化设计，有免费版 |
| Moomoo | 中等 | 有OpenAPI |
| TD Ameritrade | 好 | 被嘉信收购，未来不确定 |

### 入门建议
1. **从 Alpaca 开始** - 免费、免佣、文档友好
2. **先用模拟盘** - 至少跑几周验证策略
3. **简单策略起步** - 定投、均线突破等
